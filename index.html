Welcome on Raspberry Pi.

This site allows you to control your Raspbery Pi configured as time-lapse video sequencer.

You can:

  • Set some options (as resolution, time interval, etc.) for creating pictures.
  • Get current picture and see what is happening now.
  • Select single pictures and create video from them or delete them.

This project has been created by Jan Cernoch (CER0130) as semestral project from the subject "Multimedia data processing" on VSB - Technical University of Ostrava.
